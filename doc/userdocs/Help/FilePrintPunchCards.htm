File/Print Punch Cards...

This command prints out master punch cards for one or more courses on a printer. The punch patterns for controls should be entered by using the Controls/Punch Patterns command.

This dialog has several options to customize the result of printing.

Output section

This section shows which printer will be used, the paper size and orientation, and the margins. Click Change Printer to change which printer to use. Click Change Margins to change the paper size, orientation, or margins.

Courses section

This section shows which courses will have their control description sheets printed. The default setting is for all courses except "All Controls" to be printed. Click the checkboxes next to the courses you wish to print. Click All or None to quickly select or unselect all of the checkboxes at once.

Copies section

This section determines how many copies of each punch card will be printed. A new page is started before printing each additional copy of the punch cards.

Appearance section

The section determines how the punch cards appear.

Punch box size: enter the side of each square in the punch card. The default size of 18mm generally works well, but you may wish to adjust the size to fit your paper or exactly match the punch cards you are using.

Punch card layout: click this to set the the layout of the boxes on the punch card.

Other buttons

Print: This button prints the descriptions on the selected printer. The dialog box closes.

Preview: Opens a print preview dialog box that shows what will be printed.

Cancel: Closes the dialog box without printing anything.
