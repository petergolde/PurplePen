GPS/Create GPX File...

Creates GPX file with the control locations for use on an portable GPS unit or smart phone.

A GPX file is a standard file with GPS coordinates (waypoints) in it. The GPX file can be used with a portable GPS unit or smartphone to verify the locations of controls when setting them. Consult the documentation for your GPS unit to learn how to send GPX files to it.

In order to create a GPX file, the map file must be an OCAD or OpenOrienteering Mapper file, and the file must be georeferenced. Georeferencing a map sets up a mapping between coordinates on the paper, and geographic coordinates on the earth. See more information about setting up a georeferenced map in OCAD. Purple Pen supports most, but not all, coordinate systems that OCAD supports. Purple Pen does not support User Defined coordinate systems.

Important: Do not rely solely on GPS coordinate to place controls in the field! Competitors need to be able to navigate to the control location using only the map and compass, and it is the course setter's resposibility to ensure that the control is fairly set, that the control site is accurately mapped, and that the control site can be navigated to. GPS locations should be treated only as a double check to prevent errors.

This dialog has a few options to customize how the GPX File is created.

Courses section

This section shows which courses will be sent to the GPX files. All controls in the selected courses will be placed in the GPX files. Select "All controls" to transfer all controls. In addition to waypoints for the controls, a track in the GPX file will be created for each course.

Waypoints section

Name prefix: If something is typed in here, it is used to create the names of the waypoints. For example, if you have a control with code "31", and type "A" as the name prefix, the waypoint name in the GPX file will be "A31". If no name prefix is specified, the control code is used as the waypoint name.
