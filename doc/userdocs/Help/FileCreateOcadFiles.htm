File/Create OCAD Files...

This command creates OCAD files for each course that you specify. The OCAD file displays the course, overlaid on top of the map. This is most often used in order to print the courses, or do to further editing and customization which is beyond the abilities of Purple Pen. If you don't have a copy of OCAD, you can download the free OCAD version 6.

For each course chosen, an OCAD file with the name of that course will be created. The file for that course contains the objects with the course, and references the map file as a template or background map. OCAD 6, however, does not have the ability to have another OCAD map as a template. When using OCAD 6, to view or print the course and the map, you must open the map file and import the course file.

 

This dialog has several options to customize the result.

Courses section

This section shows which courses will have OCAD files created. The default setting is for all courses except "All Controls" to be printed. Click the checkboxes next to the courses you wish to print. Click All or None to quickly select or unselect all of the checkboxes at once.

For each course, an OCAD file with the name of the course is created.

Output section

File format: This shows which file format will be used. For best results, chose the version of OCAD you will be used to load the files.

File name prefix: If something is typed in here, it is prefixed onto the file name of each output file. For example, if you type "Final" in this box, then the result files would be "Final-White.ocd", "Final-Yellow.ocd", etc.

Folder section

This section determines in which folder (directory) the course files are created. There are three choices:

  • Same folder as Purple Pen file. The files are placed in the same folder as the Purple Pen (.ppen) file for this event. This is the default choice.
  • Same folder as map file. The files are placed in the same folder as the OCAD map (.ocd) file for this event.
  • Other folder. Any other folder. If you chose this, a "Select Folder..." button appears which you can click to select any folder.

Other buttons

Create: This button Creates the OCAD files. The dialog box closes.

Cancel: Closes the dialog box without creating anything.
