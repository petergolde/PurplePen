Item/Add Text Line

The Add Text Line command adds a line containing arbitrary text to a description sheet. This might be used to communicate special information to competitors that cannot be expressed with the standard description symbols. For example:

A text line is always attached to a control, start, finish, or crossing point. To add a text line, select one of these items, and chose the Item/Add Text Line... command. The Add Text Line dialog appears:

 

Type in the text line that you wish to appear. The following options control where the text appears:

Position:

  • Below {item}: the text line appears immediately below the selected item in the description sheet.
  • Above {item}: the text line appears immediately above the selected item in the description sheet.

Courses:

  • This course: the text line appears only in the description sheet for the current course. This option is not available if you are on the All Controls tab.
  • All courses with {item}: the text line appears above or below the selected item in any course in which the item appears.

Once the text line is inserted, it can be edited or deleted directly from the control descriptions pane. To change the text, click it after it is selected. To delete it, select it, then choose the Edit/Delete command.
