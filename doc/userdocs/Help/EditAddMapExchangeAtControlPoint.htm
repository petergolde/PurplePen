Item/Add Map Exchange/Map Exchange at Control Point

Adds a map exchange directly at a control point. At the control point, competitors pick up a new map that shows the remaining part of the course. The control point where the map exchange is located appears on both parts of the map.

When this command is selected, all the controls in the control collection that are not already in the current course are shown in light purple on the current course map.

  • To change a control that is already on the course into a map exchange control, click on that control circle.
  • To add an existing control to the current course, and make it a map exchange control, click on the light purple existing control.
  • To add a new control to the current course, and make it a map exchange control, click elsewhere on the map. The new control is created with a code selected according to the automatic numbering settings.

Once you have added a map exchange, the editing window allows switching between parts of the course.
