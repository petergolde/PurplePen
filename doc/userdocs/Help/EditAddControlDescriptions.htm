Item/Add Descriptions

Toolbar Button:

Adds the control descriptions onto the map. Only one set of control descriptions can appear for a given course. Any existing control descriptions on a course are removed when adding new control descriptions.

To add the control descriptions, chose the Item/Add Descriptions command. Press and hold the mouse button at the desired upper-left corner. Drag the mouse down and to the right until the course descriptions are the desired size, then release the mouse button. Once the control descriptions are added, they can be moved around by dragging them with the mouse, or resized by dragging the "handles" that appear on the corners and sides when it is selected.

When control description are added onto a course, the control descriptions appear for that course only. If you wish to have the control descriptions appear in the same location on multiple courses, select the Item/Change Displayed Courses command.

The control description can appear as a single column, or as multiple columns. To display control descriptions as multiple columns, just drag the left or right edge of the descriptions with the mouse to widen the descriptions until they display in multiple columns.

If you are displaying "All Parts" for a course with a map exchange, you cannot add a control description. You must select the part that you wish to add descriptions to, then add the descriptions to that part. If you wish the descriptions to appear in the same place in multiple parts, select the Item/Change Displayed Courses command.

You can add descriptions to "All Controls". These descriptions show all the controls in the event.

The appearance of the control descriptions (symbols, text, or both) can be changed with the Course/Properties command.
